# ClinicCal 診所常用計算機 (Clinical Calculator)

專為基層診所醫師設計的整合式臨床決策支援工具 (Clinical Decision Support System)。
整合了臨床常用的醫學公式、評估量表、風險預測模型以及健保查詢入口，旨在提升看診效率與醫療品質。

## ✨ 特色功能 (Features)

- **全科別覆蓋**：包含代謝、腎臟、肝膽、心血管、呼吸、神經、泌尿及身心科等 8 大分類，超過 30 種臨床工具。
- **病歷友善 (EMR Friendly)**：所有計算結果皆可一鍵複製為 SOAP Note 格式，方便貼入電子病歷系統。
- **臨床速查 (Cheat Sheets)**：提供 GCS、肌力分級、健保給付規範等常用表格速查。
- **離線運算**：核心計算邏輯皆在前端執行，反應快速且保護病患隱私。
- **RWD 響應式設計**：完美支援桌機、平板與手機操作。

## 🛠 技術堆疊 (Tech Stack)

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Icons**: Lucide React
